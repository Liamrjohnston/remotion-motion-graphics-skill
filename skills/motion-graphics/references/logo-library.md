# Bundled logo library

Copy these local assets into the project before rendering. They are starting points for identity, not permission to imply endorsement. Trademarks remain the property of their owners; follow current brand terms and replace with an exact user-supplied or official brand-kit asset when available.

- `assets/logos/claude-mark.svg` — user-approved Promptible production asset; monochrome Claude/Anthropic mark.
- `assets/logos/deepseek.svg` — Simple Icons curated vector, source `https://cdn.simpleicons.org/deepseek`.
- `assets/logos/google-gemini.svg` — Simple Icons curated vector, source `https://cdn.simpleicons.org/googlegemini`.
- `assets/logos/github.svg` — Simple Icons curated vector, source `https://cdn.simpleicons.org/github`.

- `assets/logos/cursor.svg` — Simple Icons curated vector, source `https://cdn.simpleicons.org/cursor`.
- `assets/logos/playwright.svg` — user-approved transparent Promptible production asset.
- `assets/logos/openai-codex-mark.png` — user-approved transparent Promptible production asset; use for Codex/OpenAI-related graphics only when it matches the current product identity.
- `assets/logos/slack.png` — user-approved transparent Promptible production asset; replace with official SVG when supplied/found.

Do not assume the library is current forever. Check an official page when identity appears changed. Do not fabricate a missing logo. Do not convert a low-quality raster or screenshot into a pretend SVG.
