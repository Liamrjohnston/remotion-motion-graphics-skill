# Approved reference mechanisms

Select by story mechanism. Inspect the bundled MP4 and contact sheet before coding.

## `openmontage-hook`

For fragmented tools/stages collapsing into one coherent production system.

- Opens tight on crowded real workflow surfaces.
- Pulls back to reveal one connected world.
- Surfaces physically collapse toward an authentic product/terminal center.
- Finished outputs appear as consequences.

## `openmontage-claude-pipeline`

For a request moving through real production stages.

- One connected pipeline, not disconnected scenes.
- Camera follows causality from input through work into output.
- Each destination is a distinct step; travel bridges them.

## `tsenta-apply-faster`

For slow manual work replaced by automation.

- Opens inside an authentic form at readable task scale.
- Action/time evidence establishes the old way.
- Camera reveals accumulated work, then authentic product actions.
- Cursor actions cause visible state changes and a scale payoff.

## `tsenta-one-click-wall`

For one action creating a large result.

- Opens tight on one authentic product action.
- Click causes the state change.
- Pullback reveals a coherent wall of related results.
- Scale is spatial, not an arbitrary chart/headline.

## `kickbacks-same-run-editor`

For an AI/code/tool action inside an authentic working surface.

- Editor/product UI is the world, not a decorative card.
- Camera follows the exact action/state changing.
- Detail and payoff remain attached to the working context.

## `smartlead-prewarmed-burn`

For a product action causing a metric/system state consequence.

- Faithful product UI supplies identity.
- Cursor launches the action.
- Verified system states change as direct consequences.
- Camera is restrained because authentic UI and causality carry the story.

## Selection

- fragmentation → consolidation: `openmontage-hook`
- sequential production workflow: `openmontage-claude-pipeline`
- manual labor → automation: `tsenta-apply-faster`
- one action → massive scale: `tsenta-one-click-wall`
- action inside real editor/tool: `kickbacks-same-run-editor`
- product action → system consequence: `smartlead-prewarmed-burn`

A new concept is allowed. Document why it fits the exact beat better, then inherit camera grammar, scale, restraint, and causal motion from the closest reference. Combine no more than two mechanisms. Novelty cannot become generic decoration.
